# 模块-炼丹

## 职责与边界

负责丹方读取、开始炼丹、单次 report 结算、停火、返还材料和炼丹相关术法加成。

## 入口接口

- `/api/game/alchemy/recipes`
- `/api/game/alchemy/start`
- `/api/game/alchemy/report`
- `/api/game/alchemy/stop`

## 关键数据与配置

- `recipes.json`
- `AlchemySystem`
- `AlchemyWorkshop`
- 玩家炼丹运行态与已学丹方

## 主要状态流转

- start 进入炼丹态
- report 按丹方和数量结算成丹/废丹/返还材料
- stop 结束炼丹态并收口本轮状态

## 互斥与依赖

- 与修炼、历练、草药采集互斥
- 依赖背包材料、炼丹术、丹炉、丹方

## reason_code 语义出口

以 `ALCHEMY_*` 为主，覆盖开始、上报、停止、材料不足、互斥阻塞等。
