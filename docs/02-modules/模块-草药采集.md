# 模块-草药采集

## 职责与边界

负责百草山采集点读取、开始采集、单轮 report 结算、停止采集，以及采集效率/成功率术法加成。

## 入口接口

- `/api/game/herb/points`
- `/api/game/herb/start`
- `/api/game/herb/report`
- `/api/game/herb/stop`

## 关键数据与配置

- `herb_points.json`
- `HerbGatherSystem`
- 草药采集术等级与加成

## 主要状态流转

- points 返回当前玩家视角下的采集点列表与实际耗时/成功率
- start 进入采集态
- report 单轮结算掉落
- stop 清理采集运行态

## 互斥与依赖

- 与修炼、炼丹、历练互斥
- 采集效率和成功率受术法等级影响

## reason_code 语义出口

以 `HERB_*` 为主，覆盖开始、上报、停止、互斥阻塞、时间非法等。

## 测试覆盖点

- `unit_test/test_herb_api.py`
- `unit_test/test_edge_cases_api.py`

## 维护注意事项

- `points` 返回的是“玩家实际生效后的配置”，不是原始配置直出
