# 模块-游戏基础与公共能力

## 职责与边界

负责游戏整包数据读取、离线奖励、排行榜、测试支持接口，以及依赖注入、锁、防作弊、初始化、日志等公共能力。

## 入口接口

- `/api/game/data`
- `/api/game/save`
- `/api/game/claim_offline_reward`
- `/api/game/rank`
- `/api/test/*`

## 关键数据与配置

- `Dependencies`
- `WriteLock`
- `AntiCheatSystem`
- `InitPlayerInfo`
- `TestAccountSeeder`

## 主要状态流转

- `game/data` 负责登录后整包游戏数据读取
- `claim_offline_reward` 根据离线时间结算灵气与灵石
- `rank` 负责排行榜基础数据
- `/test/*` 提供测试账号状态构造能力

## 互斥与依赖

- 所有写接口统一通过账号级写锁串行
- 业务接口依赖 `GameContext`
- 测试支持接口只允许测试账号访问

## reason_code 语义出口

- `GAME_*`
- `TEST_*`
- `GAME_WRITE_CONFLICT_RETRY`
