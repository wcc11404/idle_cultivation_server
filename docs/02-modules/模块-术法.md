# 模块-术法

## 职责与边界

负责术法解锁、列表读取、装备/卸下、升级、充灵、属性加成与战斗内使用。

## 入口接口

- `/api/game/spell/list`
- `/api/game/spell/equip`
- `/api/game/spell/unequip`
- `/api/game/spell/upgrade`
- `/api/game/spell/charge`

## 关键数据与配置

- `spells.json`
- 玩家 `player_spells`
- 已装备槽位 `equipped_spells`
- 术法类型与槽位限制

## 主要状态流转

- 解锁后进入玩家术法列表
- equip/unequip 修改槽位
- upgrade 依赖使用次数与灵气
- charge 扣减玩家灵气并增加 `charged_spirit`

## 互斥与依赖

- 战斗中禁止装备、卸下、升级、充灵
- 属性加成会反向影响玩家静态属性与修炼/战斗表现

## reason_code 语义出口

以 `SPELL_*` 为主，覆盖槽位上限、战斗锁定、升级条件不足、充灵成功失败等。
